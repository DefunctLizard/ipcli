# ipcli
A CLI tool that finds the location of IP addresses.

```
IP CLI 0.1.0
Grant H. <grantshandy@gmail.com>
Finds IP locations

USAGE:
    ipcli [OPTIONS] [ADDRESS]

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -s, --service <SERVICE>    Choose Geolocation API, if not set it defaults to ipapi [possible values: ipwhois, ipapi,
                               ipapico, freegeoip]

ARGS:
    <ADDRESS>    What IP address to look up, if none are selected your IP address will be chosen
```
